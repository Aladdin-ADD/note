一开始没反应过来，多看看才发现和进制转换是一样的呀。

```c
char* convertToTitle(int n) {
    int m = n;
    int len = 0;
    while (m-- > 0) {
        len++;
        m /= 26;
    }
    char *title = malloc(sizeof(char) * (len + 1));
    title[len--] = '\0';

    while (n-- > 0) {
        title[len--] = (n % 26) + 'A';
        n /= 26;
    }

    return title;
}
```
