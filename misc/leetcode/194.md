# 194. Transpose File

---

```sh
cat << EOF |
name age
alice 21
ryan 30
EOF
awk '{
    for (i=1; i<=NF; i++) {
        if (NR == 1) {
            word[i] = $i
        } else {
            word[i] = word[i]" "$i
        }
    }
}
END {
    for (i=1; i<=NF; i++) {
        print word[i]
    }
}'
```

---

NF 是总列数
NR 是当前行数

这种 AWK 解法应该是看起来比较简单的了
更多是要熟悉下变量、数组的处理

实际碰到这种需求，我肯定不会用 shell 来干就是了……
