# 27. Remove Element

---

```c
int removeElement(int* nums, int numsSize, int val) {
    int i = 0;
    while (i < numsSize) {
        if (nums[i] == val) {
            if (i == numsSize - 1) {
                break;
            } else {
                nums[i] = nums[--numsSize];
            }
        } else {
            i++;
        }
    }
    return i;
}
```
