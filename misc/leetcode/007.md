# 7. Reverse Integer

---

```js
/**
 * @param {number} x
 * @return {number}
 */
var reverse = function(x) {
    var s = Number(String(Math.abs(x)).split('').reverse().join(''));
    if (s > 2147483647) return 0;
    return (x < 0 ? -s : s);
};
```
