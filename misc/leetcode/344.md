# 344. Reverse String

---

```c
char* reverseString(char* s) {
	size_t end = strlen(s);
	if (end == 0) return s;

	char *ss = s;
	char *p = (s + end - 1);
	do {
		char c = *p;
		*p = *s;
		*s = c;
		s++;
		p--;
	} while (p - s > 0);

	return ss;
}
```
