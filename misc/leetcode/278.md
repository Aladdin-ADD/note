# 278. First Bad Version

---

开始有想到直接相加会不会溢出，但是没处理。
没想到最后真挂在溢出上了。

二分的想法应该还是很容易想到的。

---

```c
int firstBadVersion(int n) {
    int upper = n;
    int lower = 1;

    while (upper != lower) {
        n = lower + (upper - lower) / 2;
        if (isBadVersion(n)) {
            upper = n;
        } else {
            lower = n + 1;
        }
    }

    return lower;
}
```
