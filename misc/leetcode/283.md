# 283. Move Zeroes

---

现在看到这类问题，脑袋第一反应就是两指针遍历。

要求的第二点，没什么头绪

---

```c
void moveZeroes(int* nums, int numsSize) {
	int i = 0;
	for (int i = 0; i < numsSize; i++) {
		if (nums[i] == 0) {
			for (int j = i + 1; j < numsSize; j++) {
				if (nums[j] != 0) {
					nums[i] = nums[j];
					nums[j] = 0;
					break;
				}
			}
		}
	}
}
```

结尾处还能做点小优化，如果 j 已经到 numsSize 了，可以直接返回了。

---

```c
void moveZeroes(int* nums, int numsSize) {
	int non_zero = 0;
	int idx = 0;
	while (idx < numsSize) {
		if (nums[non_zero] != 0) {
			non_zero++;
			if (idx < non_zero) idx++;
		} else {
			while (idx < numsSize) {
				if (nums[idx] != 0) {
					nums[non_zero] = nums[idx];
					nums[idx] = 0;
					idx++;
					break;
				} else {
					idx++;
				}
			}
		}
	}
}
```

优化版，只有一次完整的遍历。真正的双指针实现。
