# 231. Power of Two

---

```c
bool isPowerOfTwo(int n) {
    while (n > 1) {
        if (n % 2 != 0) return false;
        n /= 2;
    }
    return (n == 1);
}
```

```c
bool isPowerOfTwo(int n) {
    return (n > 0 && (n & (n - 1)) == 0);
}
```
